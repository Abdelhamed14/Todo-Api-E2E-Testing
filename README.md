# Postman API Test Automation Project

## Project Summary

This project is a complete API Test Automation framework built using Postman. It demonstrates how to automate API testing by creating organized collections, writing JavaScript test scripts, executing automated tests with Newman CLI, generating detailed test reports, and integrating the testing workflow with Jenkins and GitHub Actions.

The project follows Continuous Integration (CI) practices by automatically executing API tests whenever changes are pushed to the repository, ensuring fast feedback and maintaining software quality.


## Technologies Used

- Postman
- JavaScript
- REST API
- Newman CLI
- Newman HTML Extra Reporter
- Git
- GitHub
- GitHub Actions
- Jenkins
- CI/CD

- ## Project Features

- Create and organize Postman Collections and API Requests.
- Write automated API test scripts using JavaScript.
- Validate HTTP status codes.
- Validate response body and business rules.
- Validate response time.
- Execute test collections using Postman Collection Runner.
- Run automated API tests using Newman CLI.
- Generate detailed HTML test reports using Newman Reporters.
  


- Integrate automated API testing with Jenkins.
- Execute automated API tests through GitHub Actions on every push and pull request.
- Support Continuous Integration (CI) and Continuous Testing workflows.
